### PhantomLink
<img src="https://github.com/user-attachments/assets/588be40b-ddb0-4fbc-b21d-30b5fa738b19" width="100%">
- Contributor to **PhantomLink** – a portable security research device inspired by tools like Flipper Zero
- Built as a multi-radio platform for wireless experimentation and security auditing
- Supports WiFi, BLE, Sub-GHz, and 2.4GHz ISM communication layers
- Designed as an extensible platform for both red team and blue team use cases
